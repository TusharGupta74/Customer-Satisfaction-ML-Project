# 📊 Customer Satisfaction Prediction using Machine Learning

## 📌 Project Overview
This project focuses on analyzing customer support data to understand key factors affecting customer satisfaction and building a machine learning model to predict CSAT (Customer Satisfaction Score).

The project includes end-to-end data analysis starting from data cleaning, feature engineering, exploratory data analysis (EDA), and machine learning model building.

---

## 🎯 Business Objective
The goal of this project is to help businesses improve customer service by identifying the factors that influence customer satisfaction and predicting future satisfaction levels.

---

## 🛠️ Technologies Used
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn
- NLTK (for text preprocessing)

---

## 🔍 Project Workflow
1. Data Cleaning & Preprocessing  
2. Handling Missing Values & Outliers  
3. Feature Engineering  
4. Exploratory Data Analysis (EDA)  
5. Text Preprocessing (NLP)  
6. Model Building  
   - Logistic Regression  
   - Decision Tree  
   - Random Forest  
7. Hyperparameter Tuning (GridSearchCV)  
8. Model Evaluation  
9. Model Saving & Loading  

---

## 📈 Model Performance
The models were evaluated using:
- Accuracy  
- Precision  
- Recall  
- F1 Score  

Among all models, **Random Forest** performed the best with higher accuracy and better generalization.

---

## 🤖 Final Model
**Random Forest Classifier** was selected as the final model due to its ability to handle complex data patterns and reduce overfitting.

---

## 📊 Key Insights
- Response time significantly affects customer satisfaction  
- Experienced agents (higher tenure) perform better  
- Faster issue resolution leads to higher CSAT scores
